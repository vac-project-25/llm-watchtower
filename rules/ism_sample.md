# ISM Rules (Subset)

- ISM-AC.001: Unique identities for all users; no shared accounts.
- ISM-AC.010: Multi-factor authentication for privileged access.
- ISM-LG.020: Centralised logging of authentication, authorisation, and admin actions.
- ISM-CR.100: Cryptography approved algorithms for data-at-rest.
- ISM-NW.050: Enforce TLS 1.2+ for all external connections.
